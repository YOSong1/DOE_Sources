# 20.07 주의사항·체크리스트와 민감 데이터 처리

사내 데이터를 외부 LLM 으로 보내기 전의 **데이터 위생** 패턴을 다룹니다 —
변수명 익명화와 "원시 데이터 대신 요약 통계만 전송" 원칙.
각 파일은 책 본문의 동일 번호 코드 블록(`# code_20_07_NN.py` 주석)과 1:1 매핑됩니다.

## 파일 구성

| 파일 | 내용 |
|------|------|
| `code_20_07_01.py` | 변수명·범주명 익명화 (`anonymize_columns`) — 매핑 사전은 로컬에만 보관 |
| `code_20_07_02.py` | 원시 데이터 전송(위험) vs 요약 통계 전송(안전) 비교 |

## 사용법

```bash
python code_20_07_01.py
python code_20_07_02.py
```
